### 1. **机器学习：线性回归的闭式解推导**
**问题**：如何推导线性回归的闭式解（解析解）？
**答案**：
1. **数学模型**：假设目标函数为 $y = X\theta + \epsilon$，其中 $X$ 是设计矩阵，$\theta$ 是参数向量。
2. **损失函数**：最小化均方误差 $J(\theta) = \frac{1}{2}(y - X\theta)^T(y - X\theta)$。
3. **闭式解公式**：对 $\theta$ 求导并令导数为零，得到 $\theta = (X^T X)^{-1} X^T y$。
**代码**：
```python
import numpy as np
X = np.array([[1, 2], [1, 3], [1, 4]])
y = np.array([5, 7, 9])
theta = np.linalg.inv(X.T @ X) @ X.T @ y
```
